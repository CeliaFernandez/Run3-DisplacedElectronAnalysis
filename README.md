# Run 3 superclusters analysis

## How to install

Recommended release for this analyzer is CMSSW_12_6_0 or later. Commands to setup the analyzer are:

```
cmsrel CMSSW_12_6_0

cd CMSSW_12_6_0/src

cmsenv

mkdir Analysis

cd Analysis

git clone git@github.com:CeliaFernandez/Run3-SuperclusterAnalysis.git

scram b -j 8
```
